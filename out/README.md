# My HTML Project

This project is a simple HTML webpage styled with CSS. It includes a reset CSS file to ensure consistent styling across different browsers.

## Project Structure

```
my-html-project
├── index.html      # Main HTML page
├── styles.css      # Custom styles for the webpage
├── reset.css       # CSS reset file
└── README.md       # Project documentation
```

## Files Description

- **index.html**: This is the main HTML page of the project. It includes the structure of the webpage and links to the CSS files for styling.
  
- **styles.css**: This file contains the custom styles for the webpage. It sets the background to black, text color to white, and uses the default serif font.

- **reset.css**: This file is a CSS reset that normalizes the default styling of HTML elements across different browsers to ensure consistency.

## How to Host the Page

1. Open a terminal and navigate to the project directory.
2. Use a simple HTTP server to host the page. You can use Python's built-in server:
   - For Python 3: `python3 -m http.server`
   - For Python 2: `python -m SimpleHTTPServer`
3. Open your web browser and go to `http://localhost:8000` to view the webpage.

## Additional Information

Feel free to modify the styles in `styles.css` to customize the appearance of the webpage.
